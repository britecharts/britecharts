heatmap

Heatmap

Reusable Heatmap API module that renders a simple and configurable heatmap chart.

Requires: module:d3-array,
Example 

let heatmap = heatmap();

heatmap
    .width(500);

d3Selection.select('.css-selector')
    .datum(dataset)
    .call(heatmap);
