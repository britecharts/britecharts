This is the line chart used on the Sales Analytics Report.

We have included a resize listener to show how the different charts respond to changes on the viewport.

Line Chart with fixed Aspect Ratio

The code for this example can be found in /demos/line.js and it calls the line chart with a fixed aspect ratio:

                lineChart
                    .aspectRatio(0.42)
                    .tooltipThreshold(400)
                    .width(containerWidth);

                container.datum(dataset).call(lineChart);

Line Chart with Fixed Height

In this case, we have set a fixed height on the chart, as we are currently doing on the Sales report:

                lineChart
                    .tooltipThreshold(400)
                    .height(300)
                    .width(containerWidth);

                container.datum(dataset).call(lineChart);
