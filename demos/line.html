Line Chart with fixed Aspect Ratio

The code for this example can be found in /demos/demo-line.js and it calls the line chart with a fixed aspect ratio:

The code

lineChart
    .aspectRatio(0.5)
    .tooltipThreshold(600)
    .width(containerWidth);

container.datum(dataset).call(lineChart);

Line Chart with a single data source

This is the same chart as above, but when there is a single data source we apply a gradient to the line to give it a little flair.

This chart has a fixed height like the one below.

The code

lineChart
    .tooltipThreshold(600)
    .height(300)
    .width(containerWidth);

container.datum(dataset).call(lineChart);
