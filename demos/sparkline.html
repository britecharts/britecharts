Sparkline Chart

This is the sparkline chart used for showing trends in data over time.

We have included a resize listener to show how the different charts respond to changes on the viewport.

The code for this example can be found in /demos/demo-sparkline.js and it calls the sparkline chart:

The code

sparklineChart
    .dateLabel('dateUTC')
    .isAnimated(true)
    .duration(2500)
    .height(containerWidth / 3)
    .width(containerWidth / 2);

container.datum(dataset)
    .call(sparklineChart);
