Scatter Plot with single source and horizontal grid

The code


scatterChart
    .aspectRatio(0.7)
    .width(containerWidth)
    .circleOpacity(0.6)
    .grid('horizontal')
    .xAxisLabel('Temperature (C)')
    .margin({
        left: 60,
        bottom: 50
    })
    .yAxisLabel('Ice Cream Sales');


scatterPlotContainer
    .datum(dataset)
    .call(scatterChart);
