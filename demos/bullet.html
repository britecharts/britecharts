Bullet Chart

This is the bullet chart used for showing CPU usage.

We have included a resize listener to show how the different charts respond to changes on the viewport.

The code


bulletChart
    .width(containerWidth)
    .height(400);

container.datum(dataset)
    .call(bulletChart);
